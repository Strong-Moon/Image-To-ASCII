# Image-To-ASCII
It converts image to ASCII art using pre-trained AI models.


# How Does It Works
When you write the path inside the main.py file, it takes the image and reads. Then it chooses the segmentation according to your choice.

Anime or People Segmentation. These two segmentations' purpose is to extract the background from an image. Then resize the image that you have entered the value. After that process, it grayscales the resized image.

This program is also used a gamma scaler. If the picture is dark, the gamma scaler automatically enlightens the image.

According to gammascaler, it gives each pixel its own ascii characters.(For instance for light pixels "@", for dark pixels ".") You can give this gamma scaler value by hand. 

After that process it asks you for do you want to save ascii art in a file. you have to write "yes" or "no". If answer is "yes", It creates a file for you. If "no" it exit from the program.

# How To Use 

Step 0 : download required libraries
```
python -m pip install -r requirements.txt
```

Step 1 : Enter the image path that you want to make ascii art.

Step 2 : Run the code 
```
python main.py
```
Step 3 : Select your choices.

Step 4 : Have Fun...
